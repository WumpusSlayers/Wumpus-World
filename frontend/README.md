
# 👾 Wumpus World AI -  Simulation Client

Wumpus World AI 에이전트 시뮬레이션의 실시간 격자 지도와 KB(지식 베이스) 추론 스냅샷을 시각화하고 제어하기 위한 프리미엄 웹 인터페이스 클라이언트입니다.

본 프로젝트는 별도의 빌드 단계 없이, 웹 표준 브라우저에서 바로 동작하는 가볍고 직관적인 **순수 SPA(Single Page Application)** 구조로 구성되어 있습니다.

---

## 📌 1. 실행 전 필수 체크사항 (Prerequisites)

1. **백엔드 서버 기동:**
   - 본 클라이언트는 실제 백엔드 API 서버(`http://localhost:8080`)와 연동됩니다.
   - 먼저 백엔드(`Wumpus-World-Core`) 프로젝트 루트에서 아래 명령어로 서버를 실행해 주십시오.
     ```bash
     ./gradlew bootRun
     ```
2. **Node.js 설치 여부:**
   - 본 가이드는 컴퓨터에 Node.js(npm)가 설치되어 있는 환경을 기준으로 작성되었습니다.

---

## 🚀 2. 프론트엔드 실행 방법 (Node.js `npx` 구동)

터미널 명령어 한 줄로 즉시 로컬 웹 서버를 구동하여 브라우저로 연동하는 표준적인 실행 방법입니다.

1. **터미널**을 열고 프론트엔드 프로젝트 루트 폴더(`Wumpus-World-Client`)로 이동합니다.
   
2. 아래 명령어를 실행하여 웹 서버를 띄웁니다.
   ```bash
   npx serve dist
   ```
   *(최초 실행 시 설치 동의 메시지 `Need to install the following packages: serve ... Ok to proceed? (y)`가 뜨면 **`y`**를 입력하고 엔터를 누릅니다.)*
3. 웹 서버 구동이 완료되면, 터미널 콘솔창에 표시되는 로컬 서버 주소(기본: **[http://localhost:3000](http://localhost:3000)**)로 브라우저를 통해 접속합니다.

---

## 🕹️ 3. 주요 제어 및 관전 가이드

1. **게임 시작:**
   - 좌측 컨트롤러 패널에 에이전트 이름(UserId)을 입력한 뒤 **게임 시작 🚀** 버튼을 클릭합니다.
2. **AI 자동 시뮬레이션:**
   - **`1턴 자동 실행 ⚡`**: AI 에이전트가 다음 행동 하나를 결정해 한 걸음 전진합니다.
   - **`에이전트 자동 실행 🤖▶️`**: 1.2초 간격으로 에이전트가 완수(성공 혹은 사망)할 때까지 루프를 돌며 시뮬레이션을 전 자동 관전합니다.
3. **사격 이벤트 제어 (Pause & Resume):**
   - 자동 실행 도중 에이전트가 웜파스 괴물을 향해 화살을 쏘면 **사격 전용 알림 팝업 모달창**이 팝업되며 자동 플레이가 **일시정지**됩니다.
   - 명중 결과를 확인하고 모달창의 **"확인 👍"**을 클릭하면, 멈췄던 자동 실행 루프가 **자동으로 재개**됩니다.
4. **수동 액션 조작:**
   - `전진`, `좌회전`, `우회전`, `금 획득`, `사격`, `탈출` 버튼을 통해 에이전트를 직관적으로 수동 조작할 수 있습니다.
5. **실시간 더블 맵 대조:**
   - **실시간 격자 지도:** 에이전트의 현재 각도/위치와 방문 지역, 그리고 추론을 통해 **안개속에 100% 확정한 구덩이(`🕳️⚠️`) 및 웜파스(`👾⚠️`)** 헬퍼 표시를 조명합니다.
   - **KB 추론 실시간 스냅샷:** 에이전트가 격자 단위로 유추한 실시간 기호 스크린샷을 대조하며 머릿속 상태를 모니터링합니다.
